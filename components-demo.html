details/summary 9. Accordion

O que é auy-ui?
Design system acessível com CSS classless (estiliza HTML nativo) + Web Components (Lit) para interatividade.
Precisa de JavaScript?
Não para o básico! O CSS funciona standalone — os web components são upgrade progressivo.
Quantos temas existem?
5 temas de cor (azure, emerald, ruby, amber, violet) com dark mode automático em cada um.
Suporta SSR?
Sim! Todos os componentes funcionam com Server-Side Rendering. O HTML é renderizado no servidor e os Web Components são hidratados no cliente.
